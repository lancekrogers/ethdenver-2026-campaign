# Festival Overview: grant-submissions

## Problem Statement

**Current State:** Obey Agent Economy has production-quality code across 4 blockchains with 350+ passing tests, but zero grant revenue. On-chain evidence exists for Ethereum Sepolia (CRE risk router) but NOT for 0G Galileo or Base Sepolia. Repos are private. No social media presence showing the work. Grant programs exist with active funding but we haven't applied.

**Desired State:** Submitted applications to 0G Guild ($40K ask) and Base Builder Grants (1-5 ETH) with verified on-chain transaction evidence, public repos, and social visibility.

**Why This Matters:** These are the two highest expected-value grant opportunities. 0G has a $88.88M fund and we use all 4 of their services. Base gives retroactive grants for shipped code and we integrate 3 Base-native standards. Both are realistic paths to funding within 2-6 weeks.

## Scope
