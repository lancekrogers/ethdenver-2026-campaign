# Research Findings

Registry of findings from research.

| ID | Finding | Source | Confidence |
|----|---------|--------|------------|
| | | | |

## Confidence Levels

- `high` — Well-documented, multiple sources agree
- `medium` — Documented but not fully verified
- `low` — Single source or extrapolated

## Create SUMMARY.md

After gathering findings, create `SUMMARY.md` with:
- Key conclusions
- Recommendations
- Remaining questions

## Findings Template

For significant findings, create individual files:

```markdown
# F001: [Finding Title]

**Source:** [Where this came from]
**Confidence:** high | medium | low

## Finding

[What was discovered]

## Evidence

[Supporting details, quotes, references]

## Implications

[What this means for the project]
```
