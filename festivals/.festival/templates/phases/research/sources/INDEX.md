# Research Sources

Track sources consulted during research.

| Source | Type | Status | Notes |
|--------|------|--------|-------|
| | | pending | |

## Status Values

- `pending` — Not yet reviewed
- `reviewed` — Read and documented
- `unavailable` — Could not access

## Source Types

- **documentation** — Official docs, guides, API references
- **codebase** — Existing code to analyze
- **article** — Blog posts, tutorials, papers
- **tool** — Software or services to evaluate
- **expert** — Person or team with knowledge
- **example** — Sample projects, demos, templates

## Adding Sources

```markdown
| Example API docs | documentation | pending | Main reference for integration |
| competitor-lib repo | codebase | pending | Compare implementation approaches |
| RFC 1234 | article | pending | Protocol specification |
```
