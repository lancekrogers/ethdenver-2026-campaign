# Workflow

Development workflow artifacts and processes.

## What Goes Here

- Code reviews and feedback
- CI/CD pipeline definitions
- Design documents and specifications
- Future work items (intents)

## Structure

```
workflow/
├── code_reviews/       # Code review notes and feedback
├── pipelines/          # CI/CD pipeline definitions
├── design/             # Design documents
└── intents/            # Future work items
```

## Navigation

```bash
cgo w                   # Jump to workflow/
cgo cr                  # Jump to code_reviews/
cgo pi                  # Jump to pipelines/
cgo de                  # Jump to design/
cgo i                   # Jump to intents/
```
