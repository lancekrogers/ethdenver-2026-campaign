# Codex Research

Independent validation of the existing monetization and grant research for Obey Agent Economy.

## Scope

This directory reviews:

- `workflow/explore/monetization/`
- `workflow/explore/grant-research/`

The goal is not to replace those explorations. The goal is to pressure-test them:

- Which claims are still valid
- Which claims are now stale or overstated
- Which monetization paths are most realistic given the current proof gaps

## Files

| File | Purpose |
|------|---------|
| `research-validity-review-2026-03-12.md` | Audit of what holds up, what conflicts internally, and what needs correction |
| `recommended-monetization-plan-2026-03-12.md` | Practical recommendation for how to make money from the project without over-relying on unproven pieces |
