# fest understand Topics

Reference for all `fest understand` subcommands. Run these to learn the methodology just-in-time.

## Core Topics (Start Here)

| Command | What You Learn |
|---------|---------------|
| `fest understand methodology` | Core principles, goals, and philosophy |
| `fest understand structure` | 3-level hierarchy: Phases > Sequences > Tasks |
| `fest understand tasks` | When to create task files, what they contain |
| `fest understand rules` | Naming conventions required for automation |
| `fest understand checklist` | Quick validation checklist before starting |

## Workflow Topics

| Command | What You Learn |
|---------|---------------|
| `fest understand workflow` | Just-in-time reading pattern for task execution |
| `fest understand context` | Session memory and CONTEXT.md files |
| `fest understand node-ids` | Reference system for traceability |
| `fest understand templates` | Template system for document creation |

## Advanced Topics

| Command | What You Learn |
|---------|---------------|
| `fest understand gates` | Quality gate configuration and validation |
| `fest understand plugins` | Plugin discovery and extension |
| `fest understand extensions` | Loaded methodology extensions |
| `fest understand resources` | Available methodology resources |

## Recommended Learning Order

1. `fest understand methodology` - Get the big picture
2. `fest understand structure` - Understand hierarchy
3. `fest understand rules` - Learn naming conventions
4. `fest understand tasks` - Know when to create task files
5. `fest understand workflow` - Learn the execution pattern
6. Everything else as needed
