# Guides

How-to guides, tutorials, and operational runbooks.

## What Goes Here

- Getting started guides
- Development setup
- Deployment procedures
- Operational runbooks
- Troubleshooting guides
- Best practices

## Structure

```
guides/
├── getting-started.md    # New developer onboarding
├── development/          # Dev environment setup
├── deployment/           # Deploy procedures
├── operations/           # Runbooks and procedures
└── troubleshooting/      # Common issues and fixes
```

## Guidelines

- Write for your audience (dev, ops, user)
- Include prerequisites and assumptions
- Test procedures before documenting
- Keep step-by-step instructions current
